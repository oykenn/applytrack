# ApplyTrack

A full-stack web application for tracking internship, OJT, and job applications.

## Features

* User Registration and Login
* JWT Authentication
* Dashboard Analytics
* Application Tracking
* Search and Filtering
* CRUD Operations
* Application Status Management

## Tech Stack

### Frontend

* Angular
* TypeScript
* HTML
* CSS

### Backend

* Node.js
* Express.js

### Database

* PostgreSQL

### Authentication

* JWT (JSON Web Tokens)
* bcrypt

### Backend

```bash

```

### Frontend

```bash
cd frontend/applytrack-client
npm install
ng serve
```
